# ⚠️ 重要更正：前端访问地址

## 正确的访问地址

**前端服务地址**: http://localhost:5173  
**渠道管理页面**: http://localhost:5173/channels

❌ **错误地址**: http://localhost:3001  
✅ **正确地址**: http://localhost:5173

---

## 原因说明

前端使用 Vite 开发服务器，默认端口是 **5173**，不是 3001。

---

## 快速验收指南

### 1. 打开浏览器访问

```
http://localhost:5173/channels
```

### 2. 应该看到

- ✅ 渠道管理页面
- ✅ 3个统计卡片（渠道总数：6，总订单数：100，已完成订单：21）
- ✅ 搜索框
- ✅ 渠道列表表格（6行数据）

### 3. 可以测试的功能

#### (1) 搜索功能
- 在搜索框输入 "水果" → 筛选出 "水果惠"
- 输入 "微信" → 筛选出 "刘天佟 微信" 和 "群华华 微信"

#### (2) 查看订单明细
- 点击任意渠道行的 "外部链接" 图标
- 跳转到该渠道的订单明细页面
- 应该看到该渠道的所有订单

#### (3) 修改渠道名称
- 点击任意渠道行的 "编辑" 图标（铅笔）
- 在弹窗中输入新名称
- 点击 "确认修改"
- 列表自动刷新

## 如果无法访问

### 检查服务状态

```bash
# 检查前端服务
lsof -i :5173

# 检查后端服务
lsof -i :3000

# 查看前端日志
tail -f /tmp/frontend.log
```

### 重启前端服务

```bash
# 停止前端
pkill -f vite

# 重新启动
cd frontend && npm run dev
```
